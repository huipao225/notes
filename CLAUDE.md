# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is a personal reference/notes repository containing C language learning materials and Claude Code setup guides. There is no build system, no source code to compile, and no tests to run.

## Contents

- **ClaudeCode.md** — Guide for setting up Claude Code on Windows with DeepSeek API as the backend (Chinese)
- **沟槽的c语言.md** — C language reference covering C99 standard: headers, I/O, arrays, strings, pointers, structs, linked lists, trees, functions, syntax, and sorting algorithms (Chinese)

## Conventions

- All notes are written in Chinese
- C code examples target C99 standard (`gcc -std=c99`)

## 语言要求

- 所有对话使用中文。
- 所有文档使用中文。
- 所有代码注释使用中文。

## 执行要求

- 在生成说明、总结、计划、提交说明时，统一使用中文。
- 在新增或修改 Markdown 文档时，统一使用中文。
- 在新增或修改代码注释时，统一使用中文。
